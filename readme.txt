A plugin to take appointments from the user for a date and time

== Description ==

This plugin enables take appointment feature in a wordpress website. This version of the plugin contain following features

*  Can schedule appointments weekly
*  Can have n number of time schedules in a day
*  Block specific dates from taking enquires you can set maximum 10 date ranges
*  set appointment alert email , CC, Bcc

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload 'aw-appointments' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'AW Appointments' menu in WordPress
